## Problem Statement
Leetcode link: https://leetcode.com/problems/find-minimum-operations-to-make-all-elements-divisible-by-three/description/

---
## Approach
We can just add one or subtract one.
For 3, remainder can only be 1,2
If remainder is 1 then subtract 1, else add 1.
Therefore
If remainder is not zero, then increment the count

---
## Code
```cpp
class Solution {
public:
    int minimumOperations(vector<int>& nums) {
        int n = nums.size(),count=0;
        // If remainder is 1 then subtract one
        // If remainder is 2, then add one
        // If remainder is 0, then no operation,
        // just find number of items where remiander is not zero and add one ops in it.
        for(int i=0;i<n;i++) {
            if(nums[i]%3!=0)
                count++;
        }
        return count;
    }
};
```
---
## Time complexity
O(n)
